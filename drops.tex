\section{26 - Notify parent class when
subclassing}\label{notify-parent-class-when-subclassing}

The dunder class method \texttt{\_\_init\_subclass\_\_} can be used to
notify a class when it's subclassed. This is effective for some
metaprogramming without having to resort to metaclasses.

In this example, the class \texttt{ParentCls} will print a message
whenever it is subclassed:

\begin{Shaded}
\begin{Highlighting}[]
\KeywordTok{class}\NormalTok{ ParentCls:}
    \KeywordTok{def} \FunctionTok{\_\_init\_subclass\_\_}\NormalTok{(cls, }\OperatorTok{**}\NormalTok{kwargs):}
        \BuiltInTok{print}\NormalTok{(}\SpecialStringTok{f"}\SpecialCharTok{\{}\NormalTok{cls}\SpecialCharTok{\}}\SpecialStringTok{ created with }\SpecialCharTok{\{}\NormalTok{kwargs }\OperatorTok{=} \SpecialCharTok{\}}\SpecialStringTok{"}\NormalTok{)}
\end{Highlighting}
\end{Shaded}

The argument \texttt{cls} will be the subclass, and the keyword
arguments \texttt{kwargs} come from the subclass definition:

\begin{Shaded}
\begin{Highlighting}[]
\KeywordTok{class}\NormalTok{ ChildCls(ParentCls, example}\OperatorTok{=}\VariableTok{True}\NormalTok{):}
    \ControlFlowTok{pass}
\end{Highlighting}
\end{Shaded}

When the class \texttt{ChildCls} is created, the parent class
automatically prints the following:

\begin{verbatim}
<class '__main__.ChildCls'> created with kwargs = {'example': True}
\end{verbatim}
